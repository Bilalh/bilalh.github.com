Enchanted Bashmarks - terminal directory bookmarks

Posted on January 14, 2012 in Unix Bash Scripting

Bashmarks is a shell script that allows you to save and jump to commonly used directories.

Usage Example

Liquid error: No such file or directory - posix_spawnp

Extra Features

  • if no bookmark name is specified the default directory is used instead ($HOME unless changed)

Liquid error: No such file or directory - posix_spawnp

  • Commands can be placed after the bookmark
    Liquid error: No such file or directory - posix_spawnp

  • cd like features

Liquid error: No such file or directory - posix_spawnp

Mac Specific Features

  • o command which opens the specified bookmark in Finder
  • t command which opens the specified bookmark in a new Terminal tab.

Liquid error: No such file or directory - posix_spawnp

Install

  1. git clone git://github.com/Bilalh/bashmarks.git
  2. make install
  3. source ~/.local/bin/bashmarks.sh from within your ~.bash_profile or ~/.bashrc

OR

download https://github.com/Bilalh/bashmarks/blob/master/bashmarks.sh and source it from within your ~.bash_profile or ~/.bashrc
